Basic Calculator

By: Sanchit Khope

Explanation of the Project:

This was a basic calculator that I made by myself in 2020, as my first project with Python. This code basically asks you to input two numbers and select the type of arithmetic calculation you would like to perform. If you provide an invalid input, the program handles it and prompts you accordingly.

Code:


print("Welcome to my Calculator")
while True:
    number_1 = float(input("Enter Number 1: "))
    number_2 = float(input("Enter Number 2: "))
    art = input("What do you want to do (A,S,M,D): ")

    if art == "A":
        print("Your answer is: ", number_1 + number_2)
    elif art == "S":
        print("Your answer is: ", number_1 - number_2)
    elif art == "M":
        print("Your answer is: ", number_1 * number_2)
    elif art == "D":
        print("Your answer is: ", number_1 / number_2)
    else:
        print("Your input was not valid. Please try again.")
        x = input("Would you like to try again or leave: ")
        if x.lower() == "stay":
            continue
        else:
            break
